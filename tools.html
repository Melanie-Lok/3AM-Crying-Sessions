Resources and Tools

Visual Studio Code (link)

Visual Studio Code is a free source-code editor created by microsoft, with features such as a marketplace, embedded gits and autocompletion of a piece of code.

We decided to use Visual Studio Code because we all have used it before in previous subjects and we are experienced with its features.It is accessible by most operating system (Windows, Linux, and MacOS) which ensures that all of our members are able to access it without having to worry about the compatibility of their devices. The marketplace also gave us an advantage, especially with the online server and HTML preview, making it easier to edit our HTML files.We downloaded the latest version and the tool itself is free. An alternative to Visual Studio Code is notepad++ (link)

Appcelerator (link)

Appcelerator is an open-source software development kit that allows for crossing/-platform mobile development (Android, iPhone, Blackberry, mobile web) using JavaScript as its programming language.

We would be using this app because it allows for cross-platform deployment for most of the devices elders might have. Appcelerator has a free version with plenty of features, although they have paid versions which we do not require at this moment as they are more focused on mobile businesses and companies. An alternative could have been NetBeans. (link)

Trello (link)

Trello is a free tool for collaboration and organization of projects and helps users to keep data on boards.It helps with users to tell who is working on what part of the project and what is in progress.

We use Trello to organize our project and share any important links that might be useful for our research. Trello has useful features such as checklists and labelling so each member knows what they’re working on, and figure out how long they have till the due date of each part of the project.

Alternative: Monday.com(link)

GitHub (link)

Github is a free collaborative tool that helps with the version control of our website’s development. This ensures that we are up-to-date with all our progress, and gives us a back-up of our previous versions in case we run into any issues through our development.

Our group decided to use Github because we were introduced to it in our intro to IT class so we are aware of its features, and we are able to use it without having any issues. Most developers also use github so it is easy to troubleshoot any problems that we might face.

Alternative: Bitbucket (link)

AWS(Amazon Web Services) (link)

AWS(Amazon Web Services) is a secure cloud platform where the users can store databases and host websites.It also offers compute power.

We will use the latest AWS version because RMIT provided us with it for free and we also used it last semester so we know how to handle it.

Alternative: GoDaddy (link)

Google Docs (link)

Google Docs is a free tool that also helps the users organize their documents for a project and edit a document at the same time.They can produce many things like spreadsheets, surveys and slide presentations.

We decided to use google docs because we have used it in many assignments before and by now we know how to use it and its features,and also because it is very easy and very simple.We can work on something all together at the same time and make edits without affecting someone else’s work.

Alternative: Microsoft Word (link)

Google Sheets (link)

Google Sheets is a spreadsheet program, in which you can collaborate with other users in creating spreadsheets, through your computer or even your mobile phone.

We will use Google Sheets for our Gantt chart, to keep track of our weekly progress, and the percentage of the tasks required for the submission of the project.

Alternative: Microsoft Excel (link)

Discord (link)

Discord is a free app that was firstly created for gamers and one person would host a server and people could join and all communicate at the same time.Nowadays discord is used not only for gaming but for other many reasons as well,for example creating a project with a team.

We use discord because as a team we think it is very easy to use and since everyone used it before hand we always have it opened and check messages/updates daily.Discord has a feature where you can add bots in servers and they can do multiple things.For example remind you for specific dates,record meetings,or even play music.
